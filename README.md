# Exercícios – Teste Técnico (ACE Data)

Este repositório contém a solução dos **três exercícios de lógica de programação** solicitados no teste técnico para a vaga de estágio da **ACE Data**.  

---

## Descrição dos Exercícios

### **1 - Cálculo de salário horista**

O sistema calcula:

- Salário bruto  
- Salário família (conforme faixas salariais e quantidade de filhos)  
- Salário líquido  

Entradas do usuário:

- Valor da hora  
- Horas trabalhadas no mês  
- Número de filhos menores de 14 anos  

---

### **2 - Análise de sequência numérica**

O sistema:

- Lê um valor **N** (quantidade de números)  
- Lê **N** números digitados pelo usuário  
- Exibe:  
  - A sequência completa  
  - O menor valor  
  - O maior valor  

---

### **3 - Sequência de Fibonacci**

O sistema:

- Recebe um valor **N**  
- Imprime os **N primeiros números** da sequência  
- Indica se **N pertence ou não** à sequência de Fibonacci  

---
